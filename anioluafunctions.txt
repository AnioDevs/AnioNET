Special stuff for AnioNET

["GetUserID"] = Receives and returns the UserID (Only works if user has an AnioNET account)
 Location = Server-Side
 value-type = Number
 if user doesnt have an account then error("Target's userID is unknown")
 isAuthKeyRequired = no

["GetLocation"] = Receives and returns the user's location (Example: Germany)
 Location = Server-Side
 value-type = String
 isAuthKeyRequired = yes
 aliases = getloc \ userlocation \ 

["Announce"] = Creates a new announcement
 Location = Server-Side
 value-type = Both (String & Number & Special Characters)
 isAuthKeyRequired = no
 aliases = broadcast \ message

Dosya Türleri:

.ssc = Server Side Config, client'e hiçbir veri iletilmeden ayarlanan şeylerdir. Proje sahibinden başka kimse göremez (Sonradan eklenen geliştiriciler bile) kısacası .env dosyası gibidir.

.cscfg = Client Side Config, kullanıcıya ait değişkenlerin depolanmasında kullanılabilir.

.sc = Otomatik olarak, oluşturulan her dosyaya verilir ve dosya uzantısı olarak görünmez. Script dosyasıdır, içine yazılan AnioLUA kodunu sunucuda çalıştırır. ssc ile en iyi uyan dosyadır.

.csc = Client Side Script, içerisine yazılan kodu sadece o cihazda çalıştırır. cscfg ile en iyi uyan dosyadır

.auth = AnioLUA'ya özel her bir kod için gerekli dosyadır (Örn. GetLocation.auth) içinde AuthKey bulunur ve AnioLUA'ya özel teglikeli kodların çalışmasını sağlar. Eğer AuthKey yoksa veya bu dosya yoksa hata verir.

.note = txt Dosyasına benzer, içerisinde yazılar depolanabilir. Bu yazıları kullanıcı göremez, sadece geliştiriciler görebilir.

.cfg = Hem client hemde server'da bulunan bir config dosyası.

.bsc = Hem client hemde server'da çalıştırılan bir script dosyasıdır.
